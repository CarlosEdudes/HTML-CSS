Principais formatações

Negrito / Destaque

Negrito não semântico

Use a tag B para que o destaque seja feito conforme acabamos de fazer.(Não Semântica)

Negrito semântico

Use a tag STRONG para que o destaque seja feito conforme acabamos de fazer.(Semântico)

Ítalico

Ítalico não Semântico

Use a tag I para que o destaque seja feito conforme acabamos de fazer.(Não Semântica)

Ítalico Semântico

Use a tag EM para que o destaque seja feito conforme acabamos de fazer.(Semântico)

Texto Marcado

Usando marca texto em seus textos

Para se usar o marca texto use a tag MARK, e caso queira trocar a cor da sua tag MARK use (style-background:color) logo abaixo do tittle.

Texto grande e Texto pequeno

Texto grande

Para se escrever um texto grande use a Tag Big.

Texto pequeno

Para se escrever um texto pequeno use a Tag Small.

Texto deletado e Texto inserido

Texto deletado

E a forma em que riscamos o texto, como se não fosse mais importante, para isso use a Tag del.

Texto inserido

E a forma em que se sublinha o texto, dando mais importancia, para isso use a Tag INS..

Texto sobscrito e Texto subscrito

Texto sobscrito

E a forma que usamos para elevar uma palavra conforme o exemplo abaixo.
Exemplo: x20 *3 - 4

Texto subscrito

E a forma que usamos para subscrever um texto, segue o exemplo abaixo.
Exemplo: H2O
